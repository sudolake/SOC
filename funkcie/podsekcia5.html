Kvadratická funkcia

každá funkcia s predpisom f: y = ax2 + bx + c; a, b, c ∈ R ∧ a ≠ 0

grafom je parabola

a > 0

  • konvexná
  • x1, x2 – nulové body
  • V – vrchol paraboly
  • D(f) = R
  • ani párna ani nepárna ( ak vrchol leží na osi x, t.j. v1 = 0 ⇔ b = 0, je párna)
  • nie je prostá
  • nie je periodická


a < 0

  • D(f) = R
  • nie je prostá
  • nie je periodická
  • konkávna
  • ani párna ani nepárna ( párna, ak b = 0)
