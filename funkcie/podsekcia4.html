Lineárna funkcia

Lineárna funkcia – každá funkcia s predpisom f: y = ax + b; a,b ∈ R

a = 0

  • konštantná funkcia
  • D(f) = R
  • H(f) = {b}
  • nie je prostá
  • ohraničená zhora aj zdola – b
  • vo všetkých bodoch je aj maximum aj minimum
  • je párna
  • periodická (ľubovolná)

a > 0

  • D(f) = R
  • H(f) = R
  • rastúca
  • prostá
  • nie je ohraničená
  • nemá maximum ani minimum
  • nie je párna ani nepárna (ak b = 0 ⇒ nepárna )
  • nie je periodická

a < 0

  • D(f) = R
  • H(f) = R
  • klesajúca
  • prostá
  • nie je ohraničená
  • nemá maximum ani minimum
  • ani párna ani nepárna
  • nie je periodická

b = číslo, v ktorom graf funkcie pretína y-ovú os, t.j. f(b) = b a – určuje zmenu funkčnej hodnoty, ak zvýšime x o 1

Lineárna lomená funkcia

Lineárna lomená funkcia je každá funkcia daná predpisom f: , kde a,b,c,d patrí R,c =/= 0,ad =/= bc ( cb-ad =/= 0 !!)

Grafom linárnej lomenej funkcie je rovnoosá hyperbola


Vlastnosti lineárnej lomenej funkcie:

  • D(f) = (-∞;0) (0;∞)
  • H(f) = (0;∞)
  • je nepárna
  • nie je ohraničená ani zhora, ani zdola
  • je rastúca pre x (-∞;0) a x (0;∞)
  • je prostá
  • nemá maximum ani minimum
