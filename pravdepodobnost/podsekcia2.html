Javy

Pravdepodobnosti javov

  • 0 Ω 0 - nemnožný jav P(0) = 0
  • nemnožný jav = prázdna množina ∅ = 0
  • pravdepodobnosť javou A označujeme P(A), je definovaná ako súčet pravdepodobností výsledkov priaznivých javu A, čiže
  • ak má pokus m rovnako pravdepodobnostných výsledkov, tak pričom m(A) je počet výsledkov priaznivých javu A
  • pravdepodobnosť nemožného javu = 0, P(∅) = 0
    1. pravdepodobnosť nemožného javu = 0, P(∅) = 0
    2. pravdepodobnosť istého javu = 1, P(Ω) = 1
    3. pravdepodobnosť ľubovoľného javu A = 0 ≤ P(A) ≤ 1

Nezávislé javy

  • o 2 javoch hovoríme, že sú nezávislé, ak uskutočnenie jedného javu nemá vplyv na uskutočnenie či neuskotočnenie druhého javu
  • hovoríme, že javy A a B sú nezávislé, ak P(A ∩ B) = P(A) * P(B)

Výpočet pravdepodobnosti 2 navzájom sa vylučujúcich javov a nevylučujúcich javov

  • navzájom sa vylučujúce javy A a B → A ∩ B = ∅ P(A ∪ B) = P(A) + P(B)
  • navzájom sa nevylučujúce javy A a B → A ∩ B ≠ ∅ P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
